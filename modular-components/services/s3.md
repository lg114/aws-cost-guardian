# S3 — Cost Analysis Module

## Cost Signals

- Storage class (Standard, Intelligent-Tiering, Glacier, etc.)
- Total storage size per bucket
- Number of objects
- Request patterns (GET, PUT, LIST)
- Data transfer (egress)

## Common Waste Patterns

1. **Wrong Storage Class** — Infrequently accessed data in Standard storage
2. **Incomplete Multipart Uploads** — Failed uploads consuming storage
3. **Old Versions** — Object versioning without lifecycle policies
4. **Over-replicated Data** — Cross-region replication without need
5. **Large Logs** — Application logs stored in Standard instead of Glacier

## Optimization Playbook

### Storage Class Optimization
- Use S3 Intelligent-Tiering for unknown access patterns
- Move data older than 30 days to S3 Standard-IA
- Move data older than 90 days to S3 Glacier
- Delete data older than retention policy

### Lifecycle Policies
- Automate storage class transitions
- Clean up incomplete multipart uploads after 7 days
- Expire old versions after 30 days

### Request Optimization
- Use S3 Transfer Acceleration for frequent cross-region uploads
- Batch small objects into larger files where possible

## Remediation Commands

```bash
# List buckets with sizes
aws s3api list-buckets --query 'Buckets[].Name' --output text | \
  xargs -I {} sh -c 'echo -n "{}: "; aws s3 ls s3://{} --recursive --summarize 2>/dev/null | tail -1'

# List incomplete multipart uploads
aws s3api list-multipart-uploads --bucket BUCKET_NAME

# Abort incomplete uploads older than 7 days
aws s3api list-multipart-uploads --bucket BUCKET_NAME \
  --query 'Uploads[?Initiated<=`2024-01-01`].[UploadId,Key]' \
  --output text | \
  awk '{print "aws s3api abort-multipart-upload --bucket BUCKET_NAME --key "$2" --upload-id "$1}'

# Apply lifecycle policy
aws s3api put-bucket-lifecycle-configuration \
  --bucket BUCKET_NAME \
  --lifecycle-configuration '{
    "Rules": [
      {
        "ID": "MoveToIA",
        "Status": "Enabled",
        "Filter": {"Prefix": ""},
        "Transitions": [{"Days": 30, "StorageClass": "STANDARD_IA"}]
      }
    ]
  }'
```
